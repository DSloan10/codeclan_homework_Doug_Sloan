Hints and tips

  1. region may lead to many dummy variables. Think carefully about whether to include this variable or not (there is no one ‘right’ answer to this!)

  2. Think about whether each variable is categorical or numerical. If categorical, make sure that the variable is represented as a factor.

  3. We will not treat this data as a time series, so Date will not be needed in your models, but can you extract any useful features out of Date before you discard it?

  4. If you want to build a predictive model, consider using either leaps or glmulti to help with this
